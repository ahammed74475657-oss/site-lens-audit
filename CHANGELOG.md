# Changelog

All notable changes to Site Lens Audit will be documented here.

## [1.0.0] - 2026-05-31

### Added

- Local Express server.
- Vanilla HTML, CSS and JavaScript frontend.
- URL audit form.
- SEO checks for title, meta description, H1, headings, image alt text, links, canonical URL and Open Graph tags.
- Accessibility checks for `lang`, image alt text, empty controls, unlabeled inputs and heading hierarchy.
- Estimated performance checks for HTML size, scripts, CSS resources, lazy loading and viewport meta.
- Overall, SEO, accessibility and performance scores.
- JSON and Markdown export.
- URL validation, timeout, HTML size limit and private-network blocking.

### Notes

- Lighthouse is intentionally not included in this first release.
- The tool is designed for local use, not as a hardened public proxy.
